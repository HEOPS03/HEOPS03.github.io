Начало игры¶

Чтобы приступить к игре, нужно запустить приложение двойным нажатием мыши по значку игры. Далее откроется главное меню. Для начала игры нужно нажать на кнопку «Начать игру», после чего загрузиться уровень и вы сможете приступить к игре.
